This directory preserves older KiCad concepts that are no longer part of the active release stack.

Archived here:

- `OpenRX-Nano`
- `OpenRX-900`
- `OpenRX-PWM`
- `OpenRX` placeholder project
- `OpenRX-Dual-backup` misnamed / duplicate routed-project backups

Active top-level projects are now only:

- `OpenRX-Lite`
- `OpenRX-Mono`
- `OpenRX-Gemini`

These archived projects are kept for reference, parts reuse, and historical design decisions. They should not be treated as active SKU plans.
